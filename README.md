# Bash Log Analyzer

## Description
This is a menu-driven Bash script that analyzes a log file and displays useful information.

## Features
- Menu-driven interface
- Reads a log file from the user
- Counts total lines
- Counts successful logins
- Counts failed logins
- Counts error messages

## Requirements
- Linux
- Bash

## How to Run

Make the script executable:

```bash
chmod +x log-analyzer.sh
